# Features

## IMAGE → ART

Transform uploaded images into terminal-style artwork.

### Supported Formats

* PNG
* JPG / JPEG
* WEBP
* GIF
* BMP

### Rendering Modes

| Style   | Preview      |
| ------- | ------------ |
| ASCII   | `$@#%*+=-:.` |
| DOTS    | `●◉○•`       |
| BLOCKS  | `█▓▒░`       |
| BRAILLE | `⣿⣶⣤`        |
| LINES   | `╬╪━┃`       |

### Controls

* Width control from `40 → 160`
* Intensity control from `10 → 100`
* Invert rendering toggle
* Real-time rendering preview
* Drag and drop upload area
* Export as `.txt`
* Export as `.svg`

## TEXT → ART

Generate Figlet-style text banners instantly.

### Included Fonts

* Big
* Standard
* Banner
* Block
* Doom
* Shadow
* Slant
* Ghost
* Larry 3D
* Graffiti

# API

## POST `/api/convert`

```json
{
  "image": "<base64 image>",
  "type": "ascii | dots | blocks | braille | lines",
  "width": 80,
  "intensity": 50,
  "invert": false
}
```

### Response

```json
{
  "textArt": "...",
  "svg": "<svg>...</svg>"
}
```

## POST `/api/text-art`

```json
{
  "text": "HELLO",
  "font": "Big"
}
```

### Response

```json
{
  "art": "  _   _ ..."
}
```
